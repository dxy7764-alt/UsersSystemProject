账号相关

忘记登录密码怎么办？
1. 联系班主任或系统管理员，提供学号和姓名进行密码重置；
2. 重置后的初始密码为学号后6位；
3. 登录后请及时修改为自定义密码。
账号被锁定了如何解锁？
账号连续输错密码5次会被锁定，锁定时间为30分钟。如需立即解锁，请联系系统管理员（电话：010-12345678）。
如何修改个人信息？
登录系统后，进入【个人中心】-【个人信息】页面，点击"编辑"按钮即可修改联系方式等信息。学号、姓名等关键信息需联系管理员修改。

成绩相关

成绩显示有误怎么办？
1. 确认成绩公示期内（成绩发布后7天内）；
2. 联系授课老师核实成绩；
3. 填写成绩复核申请表，提交至教务处；
4. 复核结果将在3个工作日内反馈。
什么时候可以查询成绩？
期末考试结束后10-15个工作日，任课老师完成成绩录入并审核通过后，即可在系统中查询。具体时间以教务处通知为准。
绩点是如何计算的？
绩点计算公式：
课程绩点 = 课程成绩/10 - 5（成绩≥60分）
平均绩点 = ∑(课程绩点×课程学分) / ∑课程学分
例如：85分对应3.5绩点，90分对应4.0绩点。

系统使用

系统支持哪些浏览器？
推荐使用以下浏览器：
1. Chrome 80+
2. Edge 80+
3. Firefox 75+
不建议使用IE浏览器，可能存在兼容性问题。
页面加载缓慢怎么办？
1. 检查网络连接是否正常；
2. 清理浏览器缓存和Cookie；
3. 关闭浏览器插件或扩展程序；
4. 避开高峰期（如成绩发布初期）访问系统。
如何打印成绩单？
1. 进入【个人中心】-【成绩查询】页面；
2. 点击页面右上角"打印"按钮；
3. 在打印设置中选择"PDF"或直接连接打印机；
4. 建议使用A4纸打印，纵向排版。
